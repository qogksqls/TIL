## 함수

- 함수는 값이다.

> 1급 객체(first class citizen)

1. 변수에 넣을 수 있다.
2. 함수의 인자로 넘길 수 있다.
3. 함수의 return 이 될 수 있다.

### Arrow Funtion

- 함수 선언하는 방식을 간단하게 바꿔줌

```javascript
// 기존
const arrow1 = funtion (name) {
    return name + '!'
}
```

```javascript
// 화살표 함수
const arrow2 = name => name + '!'
```
