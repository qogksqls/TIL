# 반응형 웹, Media Query
- 고정폭 레이아웃
- 유동적 레이아웃
- 별도의 사이트
- 반응형 레이아웃 : 너비 조정에 따라 콘텐츠 달라짐(bootstrap)
    - 미디어 쿼리 : @media

## breakpoints
- x-small : none <576
- small : sm >=576
- medium : md >=768
- large : lg >=992
- extra large : xl >=1200
- extra extra large : xxl >=1400

# 웹 개발 가이드
- id : 구글은 id 사용 금지, NHN은 일부 약속된 레이아웃만 활용
- class 내 속성 선언 순서 : 구글은 알파벳 순, NHN은 레이아웃 관련성에 따라
- BEM(Block Element Modifiea) 방법론
    - Block
    - Element
    - Modifiea : 형태, 행동
