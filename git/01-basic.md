# CLI(Command Line Interface) : 명령을 줄 글로 치는 곳 - Git bash 
    mkdir : 폴더 생성
    touch : 문서 생성
    rm : 문서 지우기
    ls : list 모두 보여주기
    pwd(present working dir)
    cd(change dir) : home dir(Repo) 바꿔줌
    start : 파일 열기
    cd .. : 위로가기
    cd . : 지금 위치
    .<파일명> : 숨김처리
    ls -a : all 보여주세요
    *<파일명> : <파일명>인 모든 것
    ls -t : time 순서로
