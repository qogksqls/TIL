## 강조(Emphasis)

글자의 스타일링 지정

### 기울임

`*`로 감싼 글자들은 *기울어 집니다.*

### 굵게

`**`로 감싼 글자들은 **굵어집니다.**

### 취소

`~~`로 감싼 글자들은 ~~취소선이 생깁니다.~~

### 인라인 코드

백틱으로 감싼 글자들은 `코드처럼 표시됩니다.`



## 코드 블럭(Code block)

백틱 3개로 감싼 블럭은 코드 출력용입니다.

```python
def my_fuction():
	print('Hello world')
```
