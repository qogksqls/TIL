# Start(APS 응용)

####  복잡도 분석

- Big-oh : 점근적 상한

- Big-omega : 점근적 하한

- Theta: Big-oh와 Big-omega가 같은 경우 사용

#### 비트 연산

- `&`: AND 연산
- `|`: OR 연산
- `^`: XOR 연산
- `~`: NOT
- `<<`: 열을 왼쪽으로 이동
- `>>`: 열을 오른쪽으로 이동

> 엔디안(Endianness)

- 컴퓨터의 메모리와 같은 1차원의 공간에 여러 개의 연속된 대상을 배열하는 방법을 의미하며 HW 아키텍처마다 다르다.
- 빅 엔디안: 큰 단위가 앞에 나옴. 네트워크
- 리틀 엔디안: 작은 단위가 앞에 나옴. 대다수 데스크탑 컴퓨터
